# Roadmap afbeelding-hosting app

## Doel
Een eenvoudige beheerapp voor projecten waarin afbeeldingen lokaal worden gehost, slim in mappen worden geplaatst en snel te kopiëren zijn via een vaste URL.

## MVP status
- [x] React + Vite basis
- [x] shadcn UI basis
- [x] Admin login pagina
- [x] Projecten handmatig toevoegen
- [x] Directe drag-and-drop upload
- [x] Finder-structuur: projecten, mappen, bestanden
- [x] Tabel met URL kopieerknop
- [x] Preview popup
- [x] Verwijderen van bestanden
- [x] Verwijderen van mappen
- [x] CSV export
- [ ] Productie deployment

## Fase 1
Projectbasis, componentbibliotheek, backend en opslagstructuur.

## Fase 2
Bestandsbeheer per project, uploadlogica en automatische mapvorming op basis van bestandsnamen.

## Fase 3
Gebruiksgemak: preview, kopiëren van URLs, verwijderen en export naar CSV.

## Fase 4
Beveiliging, logging, back-ups en deployment.

## Volgende stap
De online deployment, wachtwoordhardening en eventuele extra workflow-wijzigingen toevoegen.
